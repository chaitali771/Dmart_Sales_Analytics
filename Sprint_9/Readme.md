# EX9: PivotTables, Charts & Data Validation

---

```
You are working as a Junior Data Analyst in the Business Intelligence Team at DMart.
After creating sales reports and dashboards, management has noticed several data quality issues:

- Wrong category names entered
- Invalid region values
- Negative quantities entered
- Duplicate Product IDs
- Incorrect dates
- Blank mandatory fields

To improve reporting accuracy, you are tasked with:

Creating management reports using PivotTables and Charts.
Implementing Data Validation controls to prevent future data entry errors.
```
