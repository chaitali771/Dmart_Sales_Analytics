# EX7: Regional Sales Analysis

---

You are working as a Junior Data Analyst in the Business Intelligence Team at DMart.
```
The leadership team reviews monthly and quarterly sales reports but prefers visual insights over raw tables and numbers.
Management wants to understand:
- Region-wise Sales Performance
- Revenue Contribution by Region
- Profit Comparison Across Regions
- Sales Trends Over Time
- Top Performing Regions
- Underperforming Regions
```

---

You need to summarize regional sales data and create meaningful charts for management reporting.

---

# Regional Analysis Task

```
| Analysis                   | Formula/Tool            |
| -------------------------- | ----------------------- |
| Region-wise Revenue        | SUMIF / Pivot Table     |
| Region-wise Profit         | SUMIF                   |
| Average Revenue per Region | AVERAGEIF               |
| Top Performing Region      | MAX                     |
| Lowest Performing Region   | MIN                     |
| Revenue Contribution %     | Revenue ÷ Total Revenue |
| Monthly Regional Trend     | Pivot Chart             |
| Region Ranking             | RANK                    |

```
