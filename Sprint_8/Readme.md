# EX8: Lookup-Based Reporting
---

You are working as a Junior Data Analyst in the Business Intelligence Team at DMart.
```
The business maintains data across multiple sheets:
Sheet 1: Sales Data
- Order_ID	Product_ID	Customer_ID	Revenue
Sheet 2: Product Master
- Product_ID	Product_Name	Category
Sheet 3: Customer Master
- Customer_ID	Customer_Name	City
Sheet 4: Region Master
- City	Region

Note: 
Management wants a single dashboard that combines information from all these sheets.
To achieve this, you must learn and apply various Lookup Functions.
```
---

What is a Lookup Function?

A Lookup Function searches for a value in one table and returns related information from another table.
